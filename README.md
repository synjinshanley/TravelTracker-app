# Travel Tracker App

Team Members
  - Synjin Shanley
  - Lauren Applegate
  - Tristian Paquette


Main Idea: Idea: TravelTracker is a travel planning application that uses an interactive map. Users can log their activities and revisit previous travel experiences.


Features
  - Plan trips
  - Schedule events
  - Log activity
  - Upload images
  - Revisit previous trips
  - Edit trip details
  - Main map will display past routes
